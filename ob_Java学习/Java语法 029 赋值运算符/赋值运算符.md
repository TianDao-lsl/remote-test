赋值运算符

基本赋值运算符：=代表将右侧的数据交给左边的变量。
int  a = 10;

复合运算符
	+=就是a+=3；相当于 a=a+3;
	-=  就是a-=4；相当于 a=a-4;
	×=就是a×=5；相当于 a=a×5;
	÷=就是a÷=6；相当于 a=a÷6;
	%=就是a-=7；相当于 a=a%7;
int a=10,b=20,c=30,d=40,e=50;
	a+=3;
	b-=4;
	c*=5;
	d/=6;
	e%=7;
	System.out.println(a);
	System.out.println(b);
	System.out.println(c);
	System.out.println(d);
	System.out.println(e);
